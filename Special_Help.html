Infobox Episode

This section provides instructions on how to use the parseInfobox function to create and format episode infoboxes. Below you will find the syntax to use, examples of different infoboxes, and how to implement them.

Function Overview

The parseInfobox function takes text input formatted with specific syntax and converts it into HTML for displaying episode infoboxes. It also supports a dynamic title feature based on external variable values.

Syntax

To use the parseInfobox function, include the infobox markup in your text following the structure below:

Infobox Markup Structure


{{Infobox Episode
| title = Example Title
| image = example.jpg
| caption = Example Caption
| season = 1
| episodeno = 1
| airdate = January 1, 2024
| writer = John Doe
| director = Jane Smith
| previous = Previous Episode Title
| next = Next Episode Title
}}

Examples

Example 1: Complete Infobox


{{Infobox Episode
| title = Bates Motel
| image = bates_motel.jpg
| caption = Bates Motel Poster
| season = 5
| episodeno = 10
| airdate = April 30, 2017
| writer = Carlton Cuse
| director = Kerry Ehrin
| previous = The Body
| next = The Goodnight
}}

Example 2: Missing Some Fields


{{Infobox Episode
| title = Breaking Bad
| season = 2
| episodeno = 8
| airdate = August 10, 2009
| writer = Peter Gould
| director = Thomas Schnauz
}}

Example 3: Only Title and Image


{{Infobox Episode
| title = The Simpsons
| image = simpsons.jpg
}}

Example 4: Chronology Only


{{Infobox Episode
| title = Stranger Things
| previous = Chapter Three: The Monster
| next = Chapter Five: The Flea and the Acrobat
}}

Notes

Default Title

If the title field is not specified in the infobox, the function will use the page title defined in the external variable variableName and variableValue. Ensure that you pass these variables correctly for accurate results.

Dynamic Column Display

If either the previous or next fields are missing, the infobox will only display the existing one in a single-column format.

Implementation

Include the parseInfobox function in your JavaScript code and call it with the appropriate text input and external variables. For example:


const text = `{{Infobox Episode
| title = Example Title
| image = example.jpg
| season = 1
| episodeno = 1
| airdate = January 1, 2024
| writer = John Doe
| director = Jane Smith
| previous = Previous Episode Title
| next = Next Episode Title
}}`;

const variableName = 'title';
const variableValue = 'Custom Title';

const html = parseInfobox(text, variableName, variableValue);
document.getElementById('infobox-container').innerHTML = html;

Conclusion

By following this guide, you can easily format episode infoboxes for your content. Ensure your infobox syntax is correct and use the function to generate well-structured HTML output.
